## To run this
- You must have Node.js and npm running in your terminal

## Initialize npm
```
npm init
```

## Install the npm packages

### NPM PACKAGES

_https://www.npmjs.com/package/inquirer_
```
npm i inquirer
```

_https://www.npmjs.com/package/qr-image_
```
npm i qr-image
```

## Running code

```
node index.js
```
